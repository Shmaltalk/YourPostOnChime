Your Post On Chirp

What if AI turned this message into emojis?

A question we have all asked ourselves. Well now you can find out! Enter a tweet or some text and send it along to GPT for translation into (up to 6) emojis.
See some examples.






Note: due to updates to X's (formerly Twitter) API, we can no longer support Tweets.

Results are generated by sending your provided data into a large language model (GPT3) as part of a broader prompt. The model then produces statistically-likely tokens according to the prompt, in this case emojis. The results you receive can reflect biases of the model.

Emojis, a daily journal, and a social space all in one: try Chirp
